# IMM

Método de Modelos Internos (IMM) es un enfoque propio desarrollado por la entidad financiera como alternativa al modelo estándar SA-CCR. Dado que la metodología estándar tiende a ser sumamente conservadora, suele exigir un requerimiento de capital —tanto regulatorio como económico— significativamente mayor. En contraste, el IMM permite optimizar dicha carga de capital, ya que se alinea con mayor precisión al perfil de riesgo real y a las especificidades del banco.

El objetivo de este repositorio es consolidar toda la documentación técnica, regulatoria y metodológica relacionada con el Internal Model Method (IMM). Asimismo, sirve como entorno de trabajo para el desarrollo de procesos de KDD (Knowledge Discovery in Databases), facilitando la extracción de conocimiento, minería de datos y optimización de los modelos internos de la entidad
